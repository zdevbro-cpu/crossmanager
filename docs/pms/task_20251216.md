사무실 재개발 시 세팅 가이드
- 코드: `git pull origin main`.
- DB: Postgres에 `server/migrations/20251216_pms_resource.sql` 적용(서버 기동 시 자동 실행). `pgcrypto` 필요.
- 환경: `server/.env`의 DB_HOST/DB_USER/DB_PASSWORD/DB_NAME/DB_PORT 확인.
- 서버 구동: 루트에서 `npm run server` → `/api/pms/resource/*` 활성.
- 프론트(PMS) 구동: 루트에서 `npm run pms` 또는 `cd pms && npm run dev`.
- 점검 시나리오: 설계 문서의 파일럿 테스트(성공/누락/만료/WARN/override/pending/다중 assignee)로 `/eligibility/check` 호출 검증.
